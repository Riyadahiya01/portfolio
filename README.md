## Overview

This repository contains the source code for my personal portfolio website, designed to showcase my journey as an aspiring Cloud Engineer. It highlights my technical skills, cloud projects, certifications, and hands-on experience in building modern cloud solutions.

## Key Features

- Portfolio website with dedicated project case studies.
- Dynamic project case study pages for detailed project showcases.
- Smooth animations and interactive UI powered by Framer Motion.
- Fully responsive design optimized for desktop, tablet, and mobile devices.
- Well-structured and maintainable codebase built with React and TypeScript.
- Dedicated sections for Skills, Projects, Certifications, Journey, Resume, and Contact.
- Performance-focused architecture with fast loading and smooth navigation.
- Deployed on Vercel with continuous deployment support.
